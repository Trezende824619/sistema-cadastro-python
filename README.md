# 🧾 Sistema de Cadastro de Usuários em Python

Projeto desenvolvido com o objetivo de praticar conceitos fundamentais de programação em Python, simulando um sistema simples de cadastro de usuários via terminal.

🚀 Funcionalidades

* ✅ Cadastrar usuários
* 📋 Listar usuários cadastrados
* ❌ Remover usuários
* 💾 Persistência de dados em arquivo (`usuarios.txt`)

🛠️ Tecnologias utilizadas

* Python 3

📚 Conceitos aplicados

* Estruturas de repetição (`while`, `for`)
* Estruturas condicionais (`if`, `elif`, `else`)
* Listas e dicionários
* Manipulação de arquivos (leitura e escrita)
* Tratamento de erros (`try/except`)

▶️ Como executar

1. Clone o repositório:

git clone https://github.com/seu-usuario/sistema-cadastro-python.git

2. Acesse a pasta do projeto:

cd sistema-cadastro-python

3. Execute o arquivo:

python cadastro.py

📌 Observações

Os dados são armazenados localmente em um arquivo `.txt`, simulando uma persistência simples sem uso de banco de dados.
